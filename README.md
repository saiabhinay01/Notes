# 📝 Notes App

A simple and interactive Notes application built using **HTML, CSS, and JavaScript**.

This project allows users to create, view, search, update, and delete notes through a simple web interface.

## 🚀 Features

- ➕ Add new notes
- 🔍 Search notes instantly
- 👀 View note details
- ✏️ Update existing notes
- ❌ Delete notes
- 🖱️ Interactive hover effects
- 📱 Simple and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
